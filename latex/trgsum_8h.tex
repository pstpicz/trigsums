\doxysubsection{Detailed Description}
Function prototypes for the package \char`\"{}\+Trigonemetric sums\char`\"{}. 

It contains sequential and vectorized implementations of Goertzel and Reinsch algorithms for finding trigonometric sums\+: \begin{DoxyVerb}  C(x) = b_0 cos(0) + b_1 cos(x) + b_2 cos(2x) +...+ b_n cos(nx)
  S(x) =              b_1 sin(x) + b_2 sin(2x) +...+ b_n sin(nx)
\end{DoxyVerb}

Przemyslaw Stpiczynski
